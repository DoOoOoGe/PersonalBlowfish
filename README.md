# PersonalBlowfish
A personal configuration of Blowfish basing on Hugo.

## Dependence
### Hugo
Download [Hugo](https://github.com/gohugoio/hugo) and create a new project.

### Blowfish
You should add [Blowfish](https://github.com/nunocoracao/blowfish) as a submodule of your new project. Clone it to your `./themes/blowfish` directoy by running
```
git submodule add -b main https://github.com/nunocoracao/blowfish ./themes/blowfish
```
Replace `./archetypes` as `./themes/blowfish/archetypes`.

### Use this configuration
Download this repo to the `./config/_default/` replace them if pop the confirm windows.
